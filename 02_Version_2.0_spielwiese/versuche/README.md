# Versuche

Test-HTMLs und Prototypen für Retro-Mode, Boot-Sequenzen und Terminal-Simulationen.

- **test-multi-os-terminal.html** – Win/Linux/Mac Terminal mit Röhren-Look, Power-ON, Scanlines
- **test-mac-aqua-terminal.html** – macOS Aqua (OS X 10.0) Terminal mit Pinstripes, Glas-Buttons
- **test-win95-eingabeaufforderung.html** – MS-DOS-Eingabeaufforderung (3D-Rahmen, Toolbar)
- **test-win95-boot.html** – Windows-95-Boot-Animation (boot-data.js)
- **test-linux-boot.html** – Linux-LILO/Debian-Boot-Animation (boot-data.js)
- **iphone-andere-version.html** – iPhone-Easter-Egg (Linux-Boot → Panic → DFU)

Datenquellen: `../story/content/` (z. B. boot-data.js). Beim Öffnen aus dem Dateisystem ggf. relativen Pfad zu `story/content/` anpassen.
